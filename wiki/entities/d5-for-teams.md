D5 for Teams is the team collaboration edition of D5 Render, providing enterprise-level features such as team management, asset sharing, and multi-user collaborative editing.

## Nine Core Capabilities

1. **Team Management** — Team account creation and management, member invitations, and permission control
2. **Asset Management** — Team asset library deployment (supports NAS/shared drives)
3. **[[d5-studio|D5 Studio]]** — Cloud preset sharing
4. **Multi-editing** — Collaborative editing of the same project by multiple users
5. **Comments** — Project comments and annotations
6. **D5 Showreel** — Showcase platform
7. **Data Analysis** — Team data analytics
8. **Application Integration** — App integrations
9. **SSO** — Single sign-on

## Multi-editing

- Convert projects to Worksets
- Support opening multiple worksets simultaneously
- Set layers as "visible only in this workset"
- Support searching and filtering assets
- Worksets can be converted back to regular projects

## Application Integration

- **OneDrive**: Microsoft cloud storage integration
- **SharePoint**: Microsoft collaboration platform integration
- **Dropbox**: Cloud storage integration
- **ACC**: Autodesk Construction Cloud integration

## SSO

- **Microsoft Azure SSO**: Azure AD login
- **Okta SSO**: Okta identity management login

## Installation and Usage

- Log in to Team account directly in D5 Render 2.4+, no additional installation required
- Free trial: Contact the sales team
