D5 Launcher is a standalone launcher introduced from version 2.11, replacing the original welcome page and integrating D5 Render, D5 Lite, and the D5 Sync plugin.

## Feature Modules

### Home
- Create new projects or open existing ones
- View recently saved projects
- Download and install workflow plugins
- D5 update announcements and demo scene files
- View current renderer version

### Workflow
- Integrates D5 Render, [[d5-lite|D5 Lite]], and D5 Sync plugins
- Plans to gradually integrate more D5 tools and resources
- Supports login, download, installation, and update services

### Works ([[d5-works|D5 Works]])
- Browse and download 3D assets directly
- Experience consistent with D5's built-in asset library

### Projects
- Open existing projects or create new ones
- View and access all projects
- Project merge feature (Professional edition feature)

### Settings
- D5 Launcher-related parameter settings
- Renderer-related parameters must be configured within the renderer

### Help
- Official account links, help center, and official forum
- Collect logs and view system information

### Account
- Log in to D5 account from the bottom-left corner
- View account status
