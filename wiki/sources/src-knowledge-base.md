## Product Ecosystem (All-in-One Workflow)

```
D5 Lite (Concept Phase) → D5 Render (Production Phase) → D5 Works (Asset Support)
```

### D5 Lite — AI-Native Visualization Plugin

Embedded in modeling tools (currently SketchUp), providing real-time visualization during the design process. AI Conceptualization generates 4 style options from prompts, then syncs to D5 Render. Free with Pro for advanced AI features. Requires NVIDIA RTX GPU. Roadmap: Rhino, Revit, 3ds Max support; Mac version in development.

### D5 Render 3.0 — Production-Grade Real-Time Rendering Core

3.0 new features (January 2026): Ocean system, Volumetric Clouds, Displacement Material (Beta), AI Scene Match, AI Image to 3D, Procedural Buildings, D5 Works integration.

Powered by D5 Engine: built from scratch for architectural visualization with native full path tracing, Radiance Cache, ReSTIR, and multi-layer denoising. True WYSIWYG: real-time viewport = final output.

Output: up to 16K images, 8K video (Teams), 4K@120FPS (Pro), VR (SteamVR), Panorama/Spatial/XR Tours.

### D5 Works — Spatial Design Asset Platform (Beta)

39,200+ premium assets at d5works.com. Integrated into D5 Launcher with drag-and-drop placement. AEC-ready with pre-optimized performance and fidelity.

## D5 Engine Technical Architecture

**D5 Render (current)**: Built on UE5, with deep customization for architectural visualization rendering pipeline and workflow.

**D5 Lite**: Powered by D5's proprietary engine (D5 Engine), built from scratch for spatial design.

**Roadmap**: D5 Engine will gradually be integrated into D5 Render, eventually replacing the UE5 foundation — D5's long-term direction is full independence from game engine dependencies.
