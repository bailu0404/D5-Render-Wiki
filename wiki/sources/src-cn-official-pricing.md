Source summary for `raw/cn/official/pricing.md` — the CN official subscription plan comparison page.

## Key Takeaways

### Four Subscription Tiers

| Tier | Price | Target User |
|------|-------|-------------|
| 社区版 (Community) | 免费 (Free) | Students, beginners, individual explorers |
| 专业版 (Professional) | ¥200/月 或 ¥1920/年 | Practising designers, freelancers, studios |
| 团队版 (Team) | Contact sales | Design firms, enterprise teams |
| 学生版 (Student) | 免费 (Free) | Enrolled university students (verified) |

### Feature Differentiation (Community vs Professional vs Team)

**Rendering output:**
- Image/panorama: up to 16K across all tiers
- Video: up to 4K (Community/Pro), up to 8K (Team)
- Virtual tour projects: 1 (Community), unlimited (Pro/Team)

**AI:**
- AI Design Assistant: limited uses (Community), unlimited (Pro/Team)
- Text/image generation models: 10 uses across all tiers
- More AI features: 50 uses (Community), unlimited (Pro/Team)

**Asset library:**
- Official assets: 2,100+ (Community), 16,000+ (Pro/Team)

**Cloud storage:**
- Community: limited
- Professional: 10 GB
- Team: 100 GB

**Collaboration (Team edition only):**
- Multi-user project editing
- Team asset library
- Admin dashboard
- Baidu Netdisk integration

**D5 Studio presets:**
- 5 personal presets (Community), unlimited (Pro/Team)
- Team presets: Team edition only

**Virtual tour platform:**
- 1 project (Community), full access (Pro/Team)
- Advanced sharing, password protection, anchor hotspots, media hotspots, floor plan navigation: Pro/Team

### Invoicing
Professional subscribers can request invoices (发票) via the D5 User Center → Invoice Service.

### Upgrade Path
Professional → Team: contact sales for details. Or join an existing team by invitation from a team admin.
