# What is FSR Frame Generation?

To improve live rendering frame rate performance on different hardware platforms, D5 Render now adds support for FSR frame generation.

****Enable Method:**** Enable the ''FSR Frame Generation'' feature in the menu bar to take effect and improve the real-time preview frame rate.
