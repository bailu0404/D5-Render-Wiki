# How to use Advanced Image Rendering Widget?

D5 Widget is a customized extension module of D5 Render, aiming to provide advanced features while keeping the simplicity.

Go to Menu > Preference > Widget, and you can check what you installed, switch on/off with one click to manage your widgets.
