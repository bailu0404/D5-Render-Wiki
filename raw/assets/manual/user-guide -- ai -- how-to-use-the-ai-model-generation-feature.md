# How to use the AI model generation feature?

The new AI model generation function includes two core modules: ****Text to 3D**** and ****Image to 3D****. It aims to use generative AI technology to assist designers in quickly transforming inspiration into 3D assets and release creativity.

### Text to 3D

Text generation models enable content creators to convert natural-language descriptions into 3D assets quickly. Simply enter text to generate a model that meets your needs, greatly improving the efficiency of asset acquisition.

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FHIaumVowOVV9AlijNlfl%252Fimage.png%3Falt%3Dmedia%26token%3Daf74d80d-d46b-4e0a-a855-8399e92ff226&width=768&dpr=3&quality=100&sign=6dbe7d5a&sv=2)

#### How to use

### Image to 3D

The newly launched image generation model function can intelligently analyze 2D image data, infer its spatial structure, and reconstruct it into a 3D model with rich detail in a short time.

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FEg36lrX20zQwffMAF3B1%252Fimage.png%3Falt%3Dmedia%26token%3D63bdec3b-cfd3-48b4-abf6-5d908ccf13c9&width=768&dpr=3&quality=100&sign=516de84a&sv=2)

#### 1. Upload the main-view image

The main picture is the core reference for AI to understand the structure of objects.

- Operation: Drag or click to upload a main-view for generation.
- Supported formats:`.jpg`,`.jpeg`, and`.png`
- File size: up to 10 MB

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252Fie1kQk5lhcR0XqZomZR1%252Fimage.png%3Falt%3Dmedia%26token%3De8095874-6e8f-4f16-af74-d3329e51f90b&width=768&dpr=3&quality=100&sign=f1144bf7&sv=2)

Upload images

> ⚠️ > 💡****Upload Guides:****

- The quality and clarity of the reference image has a significant impact on the generated 3D model.
- High-resolution images are recommended (preferably above 1040 x 1040 pixels).
- Images with simple backgrounds, sufficient lighting, and complete and clear display of the object are recommended.

#### 2. Multi-angle Views (optional)

For a more complete model structure, you can enable the multi-view option.

- Operation: Upload up to three pictures of the same object from different angles (such as side, back, top view, etc.).
- Principle: AI will comprehensively analyze all picture information and perform three-dimensional reconstruction from multiple angles. The inference of the generated model in the invisible area will be more accurate, and the details will be richer.
- Supported formats:`.jpg`,`.jpeg`,`.png`

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FzuVcSsyLOBfKmombyi0E%252Fimage.png%3Falt%3Dmedia%26token%3D661f4b55-d989-49d9-85dd-7718a0a80755&width=768&dpr=3&quality=100&sign=773a43ff&sv=2)

Multi-angle Views

> ⚠️ > ****💡Upload Guides:****

- All pictures should be of different angles of the same object for best results.
- Please avoid uploading pictures including multiple objects or mixed views to prevent AI confusion.
- Picture order does not affect generation results.

#### 3. Generation Settings

- ****Model Name:**** After entering a name for the 3D model to be generated, click the Generate button to start a generation task.
- ****Use a Fixed Seed:**** Using the same prompt with a fixed seed produces consistent results, making it easy to reproduce specific models.
- ****Start generating:**** After the settings are completed, click the "Generate" button to create a task.

> ℹ️ > ****Precautions****

- ****Testing:**** This feature is currently in ****Beta****. After clicking the function entrance, users can receive 10 free experience opportunities. After the points are consumed, they can earn more experience points by applying for a test qualification.
- ****Credits cost:**** Each generation task will consume a certain amount of credits.
