# What do the parameters in the Statistics represent?

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FoPmniOHshFANsJbFNWcO%252Fimage.png%3Falt%3Dmedia%26token%3D71243f83-2975-484a-a873-4f09414b16c3&width=768&dpr=3&quality=100&sign=f2450a92&sv=2)

Statistics

In "Menu" → "View", click "Menu - View - Show Statistics" to open the statistics of the resources in the scene. This includes GPU, VRAM, RAM, FPS, the number of lights in the scene, the number of models and the number of faces.

## FAQs

Currently, when using AMD graphics cards, the statistics feature does not support displaying graphics memory at this time.

Please refer to the dedicated GPU memory in the task manager.
