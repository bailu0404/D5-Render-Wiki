# What's in D5 Studio

## Effect Preset

D5 effect preset contains all parameters in the D5 scene, environment and effect panel.

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FXYY826i5JTg5RSvBQkyV%252Fimage.png%3Falt%3Dmedia%26token%3De1f25b00-f3a2-431b-85e3-cddfa8bdbd4c&width=768&dpr=3&quality=100&sign=97ed8757&sv=2)

- Switch to the ready scene in the scene list.
- Right-click on the scene or click "Create Preset" in the Studio panel.

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FSSs6wgSYuht944YmvCUB%252Fimage.png%3Falt%3Dmedia%26token%3D9bf0529c-e75d-4c3c-ba4f-24f10a2c39fa&width=768&dpr=3&quality=100&sign=7430f32&sv=2)![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FL9Lqrsmavo7mAMUqxAD9%252Fimage.png%3Falt%3Dmedia%26token%3D6dda4d06-3303-42c1-9577-afccb728c9b2&width=768&dpr=3&quality=100&sign=e3b88f52&sv=2)

- After clicking "Create Preset", a pop-up window for saving the settings will appear. Complete the settings and save.

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FM4PCDDGYrjusjnPXMivB%252Fimage.png%3Falt%3Dmedia%26token%3Df09299fc-c401-4ac2-92fd-409f43b4b542&width=768&dpr=3&quality=100&sign=33a57cb0&sv=2)![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FyZbzCWs2LZVv9YlbuNaZ%252Fimage.png%3Falt%3Dmedia%26token%3D7e239f8b-6b14-416b-9f86-09a200699f93&width=768&dpr=3&quality=100&sign=f38b7d4f&sv=2)

Note:

When saving the preset, the custom HDR and LUT files used will also be uploaded.

The save time is affected by the size of the HDR file used. The preset is stored based on the data in the current preview viewport.

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FI2KqJ1XzXIHOgVrUWFV2%252Fimage.png%3Falt%3Dmedia%26token%3Df8351c74-8615-49a4-a3d8-0200f5963a07&width=768&dpr=3&quality=100&sign=d8505071&sv=2)

- In the Studio panel, double-click on the preset file that you want to apply (the first click will download the data, and the second click will apply it). The download and loading time depend on the size of the custom HDR file in the preset.

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FWju3u5KNm1r1PYSeA9cH%252Fimage.png%3Falt%3Dmedia%26token%3Df4d8742e-3e42-402f-ba1d-ca0ae2b2778b&width=768&dpr=3&quality=100&sign=763d691&sv=2)![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FxCZC8c5PB704q8tQA3xl%252Fimage.png%3Falt%3Dmedia%26token%3D3046a832-739d-410c-bafd-951249656357&width=768&dpr=3&quality=100&sign=951e66be&sv=2)

- Data ia automatically cached, so there is no need to download it repeatedly. The check mark icon in the lower right corner of the thumbnail indicates whether the preset has been downloaded.

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FcKv3gupSagkApiL6C3GN%252Fimage.png%3Falt%3Dmedia%26token%3Dd10dbb0b-7fd3-456b-bf78-0f0b8213eb85&width=768&dpr=3&quality=100&sign=a199bfad&sv=2)

- In the image or video rendering mode, applying presets is also supported.

Note: Please update the scene after applying a preset.

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FyTYM4bYFrdtHmtjlCd34%252Fimage.png%3Falt%3Dmedia%26token%3Dd4d4a6e3-8ced-4b6c-aa52-b0ee977cef1d&width=768&dpr=3&quality=100&sign=31a89794&sv=2)

## Brush preset

D5 brush preset data contains the ****asset ID**** and ****brush parameters**** used in a brush.

After logging into your Teams account, select brush tool and you can then save the preset to My Space or Team Space in the the Assets tab.

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252F3cVvFzdxhPrgoXx92MeD%252Fimage.png%3Falt%3Dmedia%26token%3D99c38e2c-ff6c-4194-ba0e-c99b79ffef66&width=768&dpr=3&quality=100&sign=3ba8d0e&sv=2)![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FKEyffAnzZzfDd9HsH1o9%252Fimage.png%3Falt%3Dmedia%26token%3D39d57ca5-fc68-4a42-8027-b577c75e38bf&width=768&dpr=3&quality=100&sign=78895519&sv=2)

Double-click on the preset thumbnail in the Studio panel to use the brush preset, and the models in the preset will be automatically downloaded and the brush tool will be activated, allowing you to directly brush in the scene. If you need to adjust the brush parameters, simply click on the preset for adjustments.

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252Fudf1xx21EQ1ECoeyLsLj%252Fimage.png%3Falt%3Dmedia%26token%3Db0827c8e-cf1f-4001-99d9-6bfafe5f2dc1&width=768&dpr=3&quality=100&sign=5b8677c0&sv=2)

## Path Preset

D5 path preset data contains the ****asset ID**** and ****brush parameters**** used in a brush.

After selecting the drawn path model, you can view the parameters of the path in the parameter panel on the right side, and save the preset to your personal space/team space through the portal.

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FY2lRBZb5Ho2AhEvSD6EB%252Fimage.png%3Falt%3Dmedia%26token%3Df8041a95-2fc6-49e3-81c8-93b02555049d&width=768&dpr=3&quality=100&sign=a855b90b&sv=2)

Double-click the path preset to open the path tool, the asset model contained in the preset will be downloaded automatically, and you can directly draw the path in the scene.
