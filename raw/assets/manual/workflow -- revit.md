# Revit

#### Download and Installation

You can download the latest version on the official website, or download from the Welcome Page > Workflow.

After downloading, double-click the .exe file and follow the instruction to install D5 Converter-Revit.

Supported versions: ****Revit 2018.3 to Revit**** ****2026****

****Create a project:**** create an empty. drs scene and import the .rvt file from the current Revit to establish a connection with D5 Render

****Associate a project:**** associate the current .rvt file in Revit with the model file in the existing .drs scene

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-legacy-files%2Fo%2Fassets%252F-MjbN1iGmN_HBnj_fyU9%252F-MkbUvk4eQ-VafTYcg1_%252F-MkbW0FD9La6JyEjt2h1%252Fimage.png%3Falt%3Dmedia%26token%3Dbc63e37e-aa8b-47ba-9e05-7d41d3b0c8f2&width=768&dpr=3&quality=100&sign=87fd8157&sv=2)

When a model file already exists in D5 Render, the first time you synchronize it, you need to connect the synchronized model to an existing model in the resource list or create a new model.

By default, D5 Render is closed. First, you need to click![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-legacy-files%2Fo%2Fassets%252F-MjbN1iGmN_HBnj_fyU9%252F-MkYOKi_31UhwPIBIJLH%252F-MkYRSrPIkTgciLY51cP%252Fimage%2520%281%29.png%3Falt%3Dmedia%26token%3D52b5d3ce-53f8-47be-8942-e5e505c602cd&width=40&dpr=3&quality=100&sign=5159ee8d&sv=2)to start D5 Render and choose to create a project or associate a project, to active D5 Converter.

If D5 Render is already started, you need to choose to create a new model or select the imported model linkage in the pop-up window.

#### Sync

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-legacy-files%2Fo%2Fassets%252F-MjbN1iGmN_HBnj_fyU9%252F-MkYOKi_31UhwPIBIJLH%252F-MkYT1emzDsDMVcxxVaV%252Fimage%2520%283%29.png%3Falt%3Dmedia%26token%3D320a36ce-28a1-44ac-a577-0ca7ababe0b8&width=40&dpr=3&quality=100&sign=7798961c&sv=2) Synchronize function, synchronize current model and corresponding material by incremental updates, which makes it faster to read Model Files.

It is suitable for changing the material and model by using the synchronization function when adjusting the details of the scheme.
