# How to select material in the scene?

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FREkYkgvcxbdSHnCP4C0q%252Fimage.png%3Falt%3Dmedia%26token%3D86b8dbb1-e432-4541-b89e-51f2d2701223&width=768&dpr=3&quality=100&sign=e1367d3c&sv=2)

Open a model via D5 Launcher, it is locked by default. Import a model when a scene file is already opened, it is unlocked by default.

## The model is locked in Object list

Selecting the material directly in the scene allows you to edit the material.

## The model is unlocked in Object list

We need to use the Material Picker to select material.

![](https://docs.d5render.com/~gitbook/image?url=https%3A%2F%2F3611830798-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MjbN1iGmN_HBnj_fyU9%252Fuploads%252FML5Wnb9LxN4S99X37oBk%252Fimage.png%3Falt%3Dmedia%26token%3D3f6c0d3f-6bc9-4a72-9fdd-7b87945a9606&width=768&dpr=3&quality=100&sign=70d98bd3&sv=2)

## Material Picker

After selecting the Material Picker (shortcut key ****i****) to activate it, you can continuously select the materials of any model in the scene, instead of exiting the material tool state immediately after selecting.

## Material Penetration

While the Material Picker is active, press the ****"Alt"**** key to penetrate the material and select the inner layer material.

- Up to 2 layers of material penetration are supported.
