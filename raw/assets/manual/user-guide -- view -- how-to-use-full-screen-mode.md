# How to use full screen mode?

Use the ****shortcut key F9**** to hide all panels in one click and display the live window in full screen, while the window supports scaling to a smaller size.

> Note: Use ****the shortcut keys F10&F11**** to hide the left and right sidebars respectively.

- In modelling, you can work with the Sync plugin to display modelling and rendering on a single screen at the same time, to enhance work efficiency.
- In video editing, you can also display full screen with one click. Press the space bar to preview the video effect directly.
