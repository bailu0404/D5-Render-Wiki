## FAQs

> When users use D5 client version ****CN\_2.8.0.0710&EN\_2.8.0.0469****, if the HDRI in the scene is in exr format and save the environment parameter to Studio preset, it will prompt "File reading error, please check the file name".

This problem has been fixed in ****CN\_2.8.0.0716 & EN\_2.8.0.0471 (official version)****. It is recommended that users reinstall in a different path to prevent incomplete overwrite installation.

> ℹ️ > The fixed version is not actively pushed to the users with abnormal version, please use the latest version of the official website as the main.
