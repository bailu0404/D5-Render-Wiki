Prompts when downloading Demo scenes

In 2.5 version, the installation experience has been optimized and the workspace has been forced to move out (Users can also customize the workspace directory during the installation). This error will be displayed if the current Windows account does not have relevant editing permissions for this workspace directory.

Suggestion: If this prompt is displayed, you can open the current workspace location to check whether you have permission to create new documents in this path; if you don't have the relevant permission, you can run D5 as an administrator or
- ****[Versions earlier than 2.11]:****migrate the workspace to a location with editing permissions in the preference settings.****- [Versions 2.11 and later]:****migrate the workspace to a location with editing permissions in D5 Launcher-Setting.
