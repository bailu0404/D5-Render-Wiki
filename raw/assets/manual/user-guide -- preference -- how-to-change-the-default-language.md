Language

In "Menu" -> "Preference" -> "General", you can set the language of D5 interface.

- D5 now supports English, Spanish, and Chinese. The UI language change will come into effect next time D5 is activated.
- Supports installing community-provided language packs in 'Preference' > 'Language'. Click 'Install language packs' and enter the download link for the pack.
