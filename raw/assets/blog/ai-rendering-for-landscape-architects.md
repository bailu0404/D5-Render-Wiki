#### ****4.**** Use the Scatter Tool to Create Biomes

The Scatter tool allows you to create diverse biomes with ease. Whether you’re designing a residential lawn or a tropical forest, presets make it simple to get started.

****Tips for Biome Creation:****

- Layer multiple scatter presets to achieve depth. For example, use a grass scatter for the base and layer it with tree scatter presets.
- Customize scatter areas by replacing individual assets or using distribution images (e.g., logos) to define specific patterns.
- Apply ****culling effects**** to exclude areas like pathways or water features from scatter distributions.

****Example:****  
For a desert biome, use presets featuring cactus and desert shrubs. Pair this with a rocky sand texture for an arid look.
