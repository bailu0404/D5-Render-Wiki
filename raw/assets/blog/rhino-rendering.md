## ****FAQs on**** Rhino Rendering with D5 Render

### 1.Why Does D5 Sync for Rhino Show 'Unknown Error'?

There are the following reasons and solutions:

1. Rhino is unable to connect to the network. It is suggested to review the firewall settings in your Windows system and either disable the firewall or remove the block on Rhino's network access.

2. It is advisable to change the installation directory of D5 Render from a folder that has Chinese characters to a folder entirely in English.

### 2.Why Isn't D5 Sync for Rhino Appearing in Rhino After Installation?

There are the following reasons and solutions:

1. Confirm that the 'D5 Sync' is enabled in 'Tools'-'Options...'Plug-ins';

2. Enable the 'D5 Sync' toolbar in 'Tools' - 'Toolbar Layout'.

### 3.Why Is 'Reload' Grayed Out in D5 Sync for Rhino?

You can only use the 'reload' feature when you import the models directly.

- If the models synced to D5 through the D5 Sync plugin need to be reload, you can use the sync feature of the plugin.
- If you use the D5 Sync plugin to export .d5a files and import them to D5 (import directly) without changing their file path, you can use the reload feature. If you have changed the path the updated models, you need to use the 'Replace from the local folder' and manually select the model files path during update.

### 4.Why Are Textures Clearer Up Close Than Far Away in D5 Via D5 Sync for Rhino?

****Reason****: Select the 'NVIDIA Control Panel-3D Settings-Use the advanced 3D image settings', which may make modeling software/D5 show blurred textures in the farther areas.

****Solution****: The correct setting is select the 'Let the 3D application decide' instead of 'NVIDIA Control Panel-3D Settings-Use the advanced 3D image settings'.

### 5.Why Is Rhino Model Orientation Different in D5's Top View After Syncing Via D5 Sync for Rhino?

****Reason****: D5 uses Left-handed coordinate system while some other software use a right-handed coordinate system, so there will be discrepancies between them.

****Solution****: It is suggested to use the D5 Sync plugin to sync the top view perspective in modeling software. This can make the top view effect be consistent with that in the modeling software.
