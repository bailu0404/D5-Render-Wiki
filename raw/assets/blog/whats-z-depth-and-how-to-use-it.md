# What's z-depth and how to use it

**Cover image credit:** [**Filip Lekawski**](https://forum.d5render.com/t/villa-w-private-resort/14572)

‍

Z-depth is important to rendering images because it provides a way to calculate depth and distance in a scene, which is crucial for creating realistic lighting and shadow effects. Without z-depth, 3D graphics would appear flat and unrealistic. In addition, z-depth is used to create depth-of-field effects, which can make images look more natural and cinematic. Overall, z-depth is an essential component of the rendering process and plays a key role in creating stunning graphics.

Luckily, D5 Render includes the z-depth channel feature to help designers achieve better visual effects.
