## ****1. What is a Virtual Tour?****

A virtual tour is an interactive 3D experience that lets viewers digitally explore spaces—whether real estate, architectural designs, or commercial environments—as if they were there in person. With [D5 Render](https://www.d5render.com/downloading?utm_source=seo&utm_medium=blog&utm_campaign=virtual-tour), creating these immersive tours becomes effortless, thanks to its powerful real-time rendering engine and seamless integration with the Showreel platform. Unlike traditional methods, D5’s workflow allows you to design, refine, and share professional virtual tours with interactive hotspots, 3D dollhouse views, and instant cloud publishing—all in one streamlined process. Ready to elevate your presentations? Let’s dive into how D5 Render makes virtual tours not just visually stunning, but truly engaging.

When setting up tour points, the camera defaults to floor level. Adjust this in the gallery to match real-world perspectives for more natural movement.

For best results, use this feature in fully enclosed indoor spaces and set at least two tour points in each area for accurate reconstruction.

## ****3. How to share via D5 Showreel****

D5 Showreel simplifies cloud-based virtual tour sharing through links or QR codes. Publish to [D5](https://www.d5render.com/downloading?utm_source=seo&utm_medium=blog&utm_campaign=virtual-tour) Cloud projects, which store multiple tours and preserve past versions, ensuring organized, accessible distribution to clients or social media.

1️⃣ Navigate to the "****Export****" menu and select "****Publish to D5 Cloud****".

2️⃣ You'll be guided to associate your virtual tour with a cloud project. At this juncture, you can either link it to a new project or integrate it with one that's already in progress.

![virtual tour](https://cdn.prod.website-files.com/62ce5d829e01c60b7c148396/68416924b451337f61ae512b_5eecdaf48460cde5c765fbf3a5df58f6e816abcf8b65c88475b8339e1c4c2483f728f271dbcdaa6639e8703ac5556d0dfc7446a1f152f0de877ffca94264dea464457a9e5705970d133459e9f99694391d8e7b4238931cbe50b8cf4ae79e877d.png)

‍

3️⃣ After uploading is completed, you can open the cloud project.

![virtual tour](https://cdn.prod.website-files.com/62ce5d829e01c60b7c148396/6841692493a49fc25106e6b2_5eecdaf48460cde5c765fbf3a5df58f6e816abcf8b65c88475b8339e1c4c2483f728f271dbcdaa6639e8703ac5556d0de35f5c220e99f0bd938e3d75d5d6368d98c27eba203e6c5fc975f4847b89aae26794cf082b17eb550872c2bb71eaaeab.png)

‍

4️⃣ On the D5 Showreel platform, you can find the uploaded virtual tour in the corresponding cloud project.

- A cloud project could contain multiple virtual tours.
- You can share virtual tours by link or QR code.

Each upload to a D5 Cloud project creates a new tour file, preserving your past work.

![virtual tour](https://cdn.prod.website-files.com/62ce5d829e01c60b7c148396/68416924008d9ef7c953915a_5eecdaf48460cde5c765fbf3a5df58f6e816abcf8b65c88475b8339e1c4c2483f728f271dbcdaa6639e8703ac5556d0db240677c8f45437432174063c2c325f9d41a28dec60cbd046cd58b5f65d92881464b307608d8195daa575c5461c2ac1a.png)

‍

5️⃣ In the Showreel platform, you can find your uploaded roaming panorama files in the corresponding cloud project.

- Multiple roaming panorama files can be uploaded in one cloud project.
- Users can copy the sharing link or generate a QRCode poster to send to their clients or share on social media.

> ******Note******: Every time the client publishes to the D5 Cloud, a new roaming file will be generated, without overwriting the previous one.
