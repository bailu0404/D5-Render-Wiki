# 1. Real-time rendering workflow for SketchUp

Aiming to build a streamlined workflow for SketchUp users, [D5 Render](https://www.d5render.com/downloading?utm_source=seo&utm_medium=blog&utm_campaign=sketchup-render) provides a ****free**** [****livesync plugin D5 Sync****](https://www.youtube.com/watch?v=3pAcmZFofLw), allowing designers to model and render at the same time.

When the plugin is enabled, you can see how your model will look in real time. Any optimization made on the model can be quickly updated in [D5 Render](https://www.d5render.com/downloading?utm_source=seo&utm_medium=blog&utm_campaign=sketchup-render). Thus you have full control over all the stages.

### Livesync features

The livesync plugin [D5 Sync works with SketchUp](https://bit.ly/4jQGIJD) versions 2020-2024 and [D5 Render 2.5](https://www.d5render.com/downloading?utm_source=seo&utm_medium=blog&utm_campaign=sketchup-render) or later versions. You can sync ****models, materials, lights and camera views**** from SketchUp to D5 Render in real time with this plugin.

**\*D5 Render offers a direct-link plugin for SketchUp 2017-2023 as well.**

![](https://cdn.prod.website-files.com/62ce5d829e01c60b7c148396/682ec42df8562112c05e9482_63d6185fd8cef97432d91745_sketchup%25E6%258F%2592%25E4%25BB%25B6ui.png)

- ****Start****: Launch D5 Render to start livesync between SketchUp and [D5](https://www.d5render.com/downloading?utm_source=seo&utm_medium=blog&utm_campaign=sketchup-render)
- ****Livesync Start/Pause****: Pause or continue the livesync process.
- ****View Sync****: Keep the viewport of SketchUp and D5 Render consistent.
- ****Light Sync****: Sync lights set in SketchUp to D5 in real time while making them grouped or separate as you need.
- ****Scene Sync****: Import the scene list from SketchUp into [D5](https://www.d5render.com/downloading?utm_source=seo&utm_medium=blog&utm_campaign=sketchup-render).
- ****Grouped Import:**** Exporting .d5a files with Group Structure info from SketchUp
