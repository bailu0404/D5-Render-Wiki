# Real-time raytracing close to Vray for interior design | GI, SSS and dynamic assets

We're glad to tell that the updated D5 Render 2.4 takes interior designers to a new render world, in which the brand new GI, subsurface scattering materials, dynamic assets and combination with DCC software help you work in a more efficient way.

The rendering tool we use here is D5 Render, which is a free standalone real-time rendering software.
