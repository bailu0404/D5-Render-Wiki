# 如何使用地形-城市功能？

* 不管是在白模还是建筑风格的状态下我们都将支持您的以下操作
  + 建筑组整体高度的**按比例**调整
  + 单个建筑高度的调整
  + 道路组整体宽度的**按比例**调整
  + 单条道路宽度的调整
* 在白模模式下，我们还支持您进行
  + 建筑组整体颜色、贴图、不透明度的调整
  + 道路组整体颜色、贴图、不透明度的调整
* 在建筑风格模式下，我们为您预设了**通用，欧洲，透明体块**三种不同风格。同时为了确保您的自定义需求，我们针对建筑组和单个建筑都设置了可调参数
  + 在通用和欧洲风格中
    - 针对建筑组您可以通过调整随机贴图为整个建筑组随机生成新的贴图，同时可以通过商业氛围控制其中现代风格外立面的比例
    - 针对单个建筑，您可以调整该建筑的立面贴图和屋顶样式，同时您可以将心仪的立面贴图复制到别的建筑上
  + 在透明体块风格中
    - 针对建筑组您可以调整透明体块的颜色和不透明度
    - 针对单个建筑，您可以深入调节首层高度和标准层高度来控制透明体块中楼板的位置
