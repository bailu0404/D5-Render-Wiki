# 如何使用D5 Scatter（散布）

1. **创建散布后右侧边栏出现提示引导后续操作**：  
   - 使用散布预设/创建自定义散布：分别打开素材库散布预设、studio散布预设分类、散布参数面板；  
   - 从素材库添加模型：打开素材库自然分类。
2. **支持按照材质和模型两种模式拾取散布的对象，形成散布区域**  
   （点击散布后默认按材质面创建散布区域，使用快捷键 X 可切换为选择模型创建散布区域）。

> 注：使用快捷键 ctrl 可以切换减选/加选和连选，选择完成后须点击确认。

3. **散布区域创建后会显示可视化的参考点**。  
   这些参考点将控制它们半径范围内植物的生成和分布，没有参考点的区域将不会有植物生长，在右侧边栏设置中可以开启/关闭参考点显示。
4. **参考点的个数有上限，并且是根据所选的散布对象自适应计算的**。  
   这意味着模型的大小会影响稀疏点之间的间隔：模型越大，稀疏点间的距离也会相应增大。
5. **散布素材与原模型/材质分离，可在场景资源列表中单独管理**。  
   当前支持聚焦、锁定、隐藏、重命名、删除操作。
