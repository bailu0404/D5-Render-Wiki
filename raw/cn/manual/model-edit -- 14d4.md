# 如何复制模型？

## 功能概览

D5渲染器中提供了多种对场景资源进行复制的方式：

| 功能 | 快捷键 \* | 用法 |
| --- | --- | --- |
| 关联复制 | Ctrl + D | 复制出来的模型，相同部分的材质互相关联。修改其中一个模型的材质，其他材质跟随变化。 |
| 独立复制 | Alt + D | 复制出来的模型，互相独立，材质不关联。修改材质时，模型之间互不影响。 |
| 拖拽关联复制 | Shift + 拖拽坐标轴 | 沿着某一方向的坐标轴复制。复制出来的模型，材质互相关联。 |

> 快捷键支持自定义修改。

  
  

## 操作详解

### 关联复制

使用快捷键 Ctrl + D 复制时，材质默认互相关联。即调任意一个模型上的材质，另一个模型也跟随变化。

### 独立复制

如需要在复制时不关联材质，可以使用 Alt + D 复制模型。

### 拖拽关联复制

先按住 shift ，然后点击任意方向的坐标轴，可以将该模型沿着指定方向复制。
