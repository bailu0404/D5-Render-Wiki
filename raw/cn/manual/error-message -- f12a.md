# 第一次打开D5出现"由于找不到 Qt5Core.dll，无法继续执行代码。"怎么办？

![](https://saas.bk-cdn01.com/t/18217684-957c-4109-9021-5866cc58cc60/u/10a66288-31af-4b72-adee-8249db06c5f0/1678686605028/image.png)可能是以下原因导致的：

1. 下载的安装包不完整，推荐使用[Chrome浏览器](https://www.google.cn/intl/zh-CN/chrome/)或者 [edge浏览器](https://www.microsoft.com/zh-cn/edge)下载
2. 第三方解压软件解压过程中丢失文件。推荐使用Windows官方解压或者使用[7-zip](https://www.7-zip.org/)解压。
3. 没有当前D5安装的目录的相关权限导致的。在确认更改安装时，换一个磁盘安装。
