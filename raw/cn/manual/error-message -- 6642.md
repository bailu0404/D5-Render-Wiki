# 为什么会提示“此材质不可复用”？

![](https://saas.bk-cdn01.com/t/18217684-957c-4109-9021-5866cc58cc60/u/10a66288-31af-4b72-adee-8249db06c5f0/1689160046845/Dingtalk_20230712190643.jpg)一般是一下几种情况比较容易出现这个弹窗：

1. 模型位置离原点位置较远
2. 选择其他材质赋给植物
3. 材质还未加载完成就进行复用操作
4. 吸取材质后赋予材质时没有选中模型
5. 该模型为素材库模型且不支持赋予用户手动赋予材质
6. 对（显示/隐藏的）路径进行赋予材质的操作
7. 对剖切工具进行赋予材质的操作
