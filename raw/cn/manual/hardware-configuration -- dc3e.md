# 怎么使用support tool？

关键字：日志

### support tool的三种打开方式：

* 当前运行的D5安装目录下的support tool
* D5 启动器左下角—帮助（2.11及以上版本）
* 菜单—帮助—系统信息

### support tool运行方式：

点击收集日志，会自动生成`zziplog.zip`文件，此文件即为当前设备上运行该版本D5的日志信息。![](https://saas.bk-cdn01.com/t/18217684-957c-4109-9021-5866cc58cc60/u/7531f9aa-9f13-43a5-858c-19dac517bbc6/1754386282980/e3050eb764df41e21a56111785fb67dc.jpg)2025-08-05
