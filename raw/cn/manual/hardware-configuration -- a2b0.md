# CPU对D5运行影响大吗？

一般情况下，不同CPU对D5影响不大。但建议关闭CPU超频，以获得稳定性能。

---

注意我们注意到D5渲染器在  **Intel Core i9-13900K/KF/KS或i9-14900K/KF/KS CPU**上频繁崩溃的问题。 当前在这些设备运行D5及其他DX12软件时，可能会遇到闪退、崩溃、加载卡99%、“系统中部分DLL文件错误”或“程序启动失败”的弹窗信息的情况。【2024年9月更新】Intel已向OEM/ODM合作伙伴发布（0x129）微码补丁。用户可到主板官网更新BIOS版本。但基于此前问题的崩溃 可能已导致不可逆的损坏 需要更换。相关内容可参考：[`Intel Processor Instability Causing Oodle Decompression Failures (radgametools.com)`](https://www.radgametools.com/oodleintel.htm)2024-09-20
