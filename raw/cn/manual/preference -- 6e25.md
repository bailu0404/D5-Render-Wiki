# 如何查看和修改默认快捷键？

在**“菜单” -> “偏好设置” -> “快捷键”**中可自定义快捷键，根据自己的使用习惯修改键位，与常用的工作流衔接。![](https://saas.bk-cdn01.com/t/18217684-957c-4109-9021-5866cc58cc60/u/b2b089df-cb81-4043-b79c-df8b2dc9bba1/1671973582502/image.png)**自定义快捷键：**常见的操作如视角移动、视图切换、各类工具都已支持修改。还有部分功能预留了空白键位，可直接自定义。点击输入框中输入需要替换的键，按下回车键即可修改成功。设置成功后，也支持点击“还原”重置快捷键。如设置时遇到快捷键冲突，右键可以取消设置。![](https://saas.bk-cdn01.com/t/18217684-957c-4109-9021-5866cc58cc60/u/b2b089df-cb81-4043-b79c-df8b2dc9bba1/1671973593405/image.png)小键盘与大键盘的数字键做了分离，分别支持设置单独的快捷键。原小键盘数字键（1、2、3、4）不再支持光源的添加。

## D5渲染器常用快捷键

|  |  |  |
| --- | --- | --- |
| 快捷键 | 操作 | 简介 |
| W A S D Q E | 前后左右上下 | 漫游模式下的移动操作 |
| Shift+Space+W A S D | 加速/减速 | 调节漫游模式下的移动速度 |
| Alt+V | 切换环视/漫游 | 视图模式切换环视或漫游，漫游状态下可自由移动 |
| V | 切换移动旋转、缩放 | 选中模型后，三轴操作在“移动旋转”和“缩放”间切换 |
| I | 选择材质 | 激活材质吸管，吸取后物体表面后调出材质面板 |
| F1/F2 | 渲染画质精细/流畅 | 精细画质下预览效果最佳，流畅画质下部分效果减弱或不显示，以减少资源占用 |
| M | 素材库 | 打开D5渲染器内置素材库 |
| F/T/P | 前视图/顶视图/透视视角 | 切换预览视角 |
| L | 隐藏/显示全部载体 | 载体用于在场景中摆放灯光、粒子、路径 |
| C/R | 放置前缩放/放置前旋转 | 从素材库选取模型，当模型跟随鼠标时，通过快捷键调整角度和大小 |
| Z | 聚焦 | 预览画面快速聚焦到选中物体 |
| Ctrl | 多选/框选 | 按住 Ctrl 后可多选模型 |
| Ctrl+S | 保存 |  |
| Ctrl+G | 成组 |  |
| Ctrl+Z | 撤销/撤回 |  |

| ↓ | 下移选中状态 |  |
