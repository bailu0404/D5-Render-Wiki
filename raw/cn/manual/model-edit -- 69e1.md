# 如何将模型文件导入D5？

## 概述

D5渲染器主要有两种方式导入外部模型数据文件

1. 支持直接打开模型文件
2. 通过工作流插件，将建模软件和D5渲染器建立连接

除此之外，D5渲染器素材库包含多种植被、人物、室内家居、商业配景等模型，可以直接在项目中使用。
