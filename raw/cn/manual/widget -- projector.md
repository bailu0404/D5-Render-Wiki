# 如何使用投影仪/投影灯组件？

「D5组件」是基于D5渲染器简洁的本体之上，用于实现高级功能延伸的一系列扩展模块。在“菜单” -> “偏好设置” -> “组件”中，可以查看当前已有的组件，一键开关，管理组件的状态。![](https://saas.bk-cdn01.com/t/18217684-957c-4109-9021-5866cc58cc60/u/b2b089df-cb81-4043-b79c-df8b2dc9bba1/1661742680324/%E7%BB%84%E4%BB%B62.2.png)

---

在“菜单” → “偏好设置” → “组件”中，开启投影灯后，可以从“导航栏” -> “光源”，选择“投影仪”添加到场景中。![](https://saas.bk-cdn01.com/t/18217684-957c-4109-9021-5866cc58cc60/u/b2b089df-cb81-4043-b79c-df8b2dc9bba1/1661743305309/%E8%8F%9C%E5%8D%95-%E7%81%AF.png)投影仪投影效果为矩形。支持添加图片（jpg / png / bmp）、视频（mp4 / avi / wmv）格式的媒体文件。将投影仪添加至场景中后，可在右侧边栏调整投影仪参数![](https://saas.bk-cdn01.com/t/18217684-957c-4109-9021-5866cc58cc60/u/b2b089df-cb81-4043-b79c-df8b2dc9bba1/1661743319108/%E6%8A%95%E5%BD%B1.png)

* Video：可以选择预设图案或上传自定义图案确定投影画面。
* 亮度：光源的发光强度，单位为 cd（candela，坎德拉），最大亮度可输入 8000000。
* 锥角：控制投影仪发光圆锥的锥角范围，通过锥角参数可控制投影图案的反馈。
* 衰减半径：限制光源的影响范围。在此范围内，光线依照正常的平方反比定律衰减，超出此范围，停止光照计算。
* UV：控制投影内图片或视频的位置或尺寸。
* 烟雾效果：若需要模拟投影仪中烟雾参数的效果，首先需要先打开“环境” -> “天气” -> “雾气”中的“丁达尔效应”。

## 常见问题

### 1. 为什么投影灯效果异常？

当前版本，D5舞台灯/投影灯数量上限为64个，超出该使用上限可能会导致预览/输出效果异常。

### 2. 为什么镜面反射舞台灯/投影仪效果在渲染结果中不显示？

当前版本暂时还不支持此效果，会在后续版本进行相关功能的支持。![](https://saas.bk-cdn01.com/t/18217684-957c-4109-9021-5866cc58cc60/u/10a66288-31af-4b72-adee-8249db06c5f0/1707188011317/%E6%9C%AA%E6%A0%87%E9%A2%98-2.png)当前版本镜面反射舞台灯/投影仪的预览&输出效果2024-02-06
