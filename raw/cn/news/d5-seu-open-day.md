# 联结设计与未来：D5渲染器 X 东南大学建筑学院学生开放日

2025年11月12日，D5渲染器联合东南大学建筑学院共同举办了学生开放日活动。

本次活动不仅打破了高校教育与行业实践的壁垒，更搭建起未来建筑设计师与前沿技术、专业研发团队深度对话的桥梁。

![联结设计与未来：D5 渲染器 X 东南大学建筑学院学生开放日](https://cn-market.d5cdn.com/website/blog/d5-seu02.jpg)

师生们在D5团队的引导下参观了公司的工作环境。从研发部门到设计空间，同学们得以近距离感受D5的真实工作场景，对技术背后的团队与文化建立了直观的认知。

活动现场，东南大学建筑学院院长助理、博士生导师周欣老师代表校方致辞。她表示，此次走进企业是一次难得的学习机会，有助于同学们跳出课堂，了解行业前沿工具的应用与发展，并对D5为建筑教育所提供的支持表示感谢。

![联结设计与未来：D5 渲染器 X 东南大学建筑学院学生开放日](https://cn-market.d5cdn.com/website/blog/d5-seu03.jpg)

随后，D5团队成员带大家了解了公司的发展历程、AI 业务与高校合作生态，并重点展示了为高校在校生推出的特别权益计划「D5学生版」：免费开放核心功能和全部素材，助力高校教与学。

另外，学生活动也是D5长期探索的方向，包括旨在挖掘设计新秀的「D5杯高校设计大赛」、深度链接D5和高校的「校园伙伴计划」，D5团队成员也对此进行了介绍，展现了D5对学生群体从学习到实践全链路成长生态的有力支持。

![联结设计与未来：D5 渲染器 X 东南大学建筑学院学生开放日](https://cn-market.d5cdn.com/website/blog/d5-seu04.jpg)

D5的技术团队成员，则聚焦到D5产品本身，通过南京本土建筑设计案例，深入介绍了D5的产品特点和核心功能，现场同学都表示受益匪浅。

![联结设计与未来：D5 渲染器 X 东南大学建筑学院学生开放日](https://cn-market.d5cdn.com/website/blog/d5-seu05.jpg)

在职业发展交流环节，D5人力资源团队精心设计了“岗位盲盒”的互动游戏。通过任务盲盒和分组讨论的趣味形式，同学们了解了D5各团队的岗位设置和工作内容。大家纷纷表示，此次访问不仅让他们对实时渲染技术有了全新认识，更对个人职业规划有了更清晰的思考。

![联结设计与未来：D5 渲染器 X 东南大学建筑学院学生开放日](https://cn-market.d5cdn.com/website/blog/d5-seu01.jpg)

本次开放日不仅是一次简单的参观，更是一次深度的双向交流。

它让同学们直观感受到了一款渲染工具从构想变为现实的过程，也帮助D5倾听了来自未来设计力量最真实的声音。我们相信，今日的交流已在许多同学心中播下了种子，D5将继续陪伴在每一位创作者身边，共同见证更多精彩设计的诞生。
