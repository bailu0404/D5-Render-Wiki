## 主要竞品概况

## 🎯 一句话竞品总结

| 竞品 | D5的立场 |
|------|----------|
| vs **Lumion** | D5：更好的AI，真正WYSIWYG，更低价格，更丰富集成；Lumion：大气效果略强，Mac支持更好 |
| vs **Enscape** | D5：更完整的功能+AI，超越插件定位，多人协作；Enscape：BIM原生插件体验更顺畅，轻量 |
| vs **Twinmotion** | D5：专为AEC优化的AI生态，价格更低，操作更简单；Twinmotion：UE生态支撑，游戏跨界 |
| vs **V-Ray/Corona** | D5：实时+AI，效率提升10倍，学习曲线低；V-Ray/Corona：离线渲染画质最准确，3ds Max原生 |
| vs **Unreal Engine** | D5：开箱即用，专为建筑师，15分钟上手；UE：高度定制，游戏级交互，但需专业程序开发 |
| vs **通用AI（ChatGPT等）** | D5：集成工作流内，PBR材质精准，无需切换应用；通用AI：概念灵感有价值，但工作流割裂 |

## 💪 D5 绝对优势（无竞品能完全复制的组合拳）

1. **15+项 in-app AI工具**，全流程覆盖（任何竞品最多2-3项AI功能）
2. **All-in-One生态**：Lite + Render + Works，一个账号、一套工作流
3. **自研D5 Engine**，WYSIWYG真正精准，非游戏引擎改造
4. **多人实时协同编辑**（在实时渲染软件中极其稀缺）
5. **XR Tour**（Gaussian Splatting，浏览器直接访问，无需安装软件）
6. **免费Community版**（含完整功能），门槛最低
7. **社区背书**：G2 #1，世界前10大事务所80%使用

## ⚠️ D5 当前局限（诚实评估）

1. **Mac版本尚未完整支持**（D5 Lite Mac版开发中）
2. **AMD GPU光线追踪受限**（部分AI/RT功能不可用）
3. **离线渲染画质**：V-Ray/Corona最终帧质量仍略优于实时（专业高端出图场景）
4. **游戏/影视级交互**：Unreal Engine在此维度领先
5. **3ds Max/Maya原生生态**：V-Ray/Corona在该人群中更根深蒂固
